### Binlog Usage

> On the **source/primary**:


**Compute current file % usage** (manual):  
Match `SHOW MASTER STATUS.File` with `SHOW BINARY LOGS.Log_name`, then calculate:

    percent = (Position / File_size) * 100

## ðŸ”§ Helpful threshold checks (MySQL 5.5)

### Connections â‰¥â€¯85% of max

```sql
SELECT
  @@max_connections AS max_conn,
  (SELECT VARIABLE_VALUE FROM INFORMATION_SCHEMA.GLOBAL_STATUS WHERE VARIABLE_NAME='Threads_connected') AS current_conn,
  ROUND( ( (SELECT VARIABLE_VALUE FROM INFORMATION_SCHEMA.GLOBAL_STATUS WHERE VARIABLE_NAME='Threads_connected') / @@max_connections ) * 100, 2 ) AS pct_used;
```

### Buffer Pool hit ratio (approx)

```sql
SELECT
  100 - ROUND(
    ( (SELECT VARIABLE_VALUE FROM INFORMATION_SCHEMA.GLOBAL_STATUS WHERE VARIABLE_NAME='Innodb_buffer_pool_reads')
    / (SELECT VARIABLE_VALUE FROM INFORMATION_SCHEMA.GLOBAL_STATUS WHERE VARIABLE_NAME='Innodb_buffer_pool_read_requests') ) * 100
  , 4) AS buffer_pool_hit_pct;
```

### Slow queries rate (since startup)

```sql
SELECT
  (SELECT VARIABLE_VALUE FROM INFORMATION_SCHEMA.GLOBAL_STATUS WHERE VARIABLE_NAME='Slow_queries') AS slow_queries,
  (SELECT VARIABLE_VALUE FROM INFORMATION_SCHEMA.GLOBAL_STATUS WHERE VARIABLE_NAME='Uptime') AS uptime_seconds,
  ROUND(
    ( (SELECT VARIABLE_VALUE FROM INFORMATION_SCHEMA.GLOBAL_STATUS WHERE VARIABLE_NAME='Slow_queries')
    / NULLIF((SELECT VARIABLE_VALUE FROM INFORMATION_SCHEMA.GLOBAL_STATUS WHERE VARIABLE_NAME='Uptime'),0) )
  , 6) AS slow_queries_per_sec;
```
