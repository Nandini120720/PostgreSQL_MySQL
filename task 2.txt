5. Copy 1 database from one to another server with different target db name.
Example Scenario

Source server: source_host

Source database: db_source

Target server: target_host

Target database (new name): db_target

Dump the Source Database
  mysqldump -h source_host -u root -p --single-transaction --routines --triggers --databases db_source > db_source.sql

# Replace the Source DB Name with Target DB Name 
This changes every mention of db_source to db_target (usually in CREATE DATABASE and USE statements):
sed 's/`db_source`/`db_target`/g' db_source.sql > db_target.sql

# Create the Target Database (Optional but Recommended)
Before importing, ensure the target database exists:
mysql -h target_host -u root -p -e "CREATE DATABASE IF NOT EXISTS db_target;"

# Import the Modified Dump File
mysql -h target_host -u root -p db_target < db_target.sql (Now db_source is fully copied to db_target on the new server.)

# After import, verify on the target server:
SHOW TABLES FROM db_target;
